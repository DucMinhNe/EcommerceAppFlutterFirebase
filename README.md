## Overview

Ecommerce Book App is a Flutter application that lets shoppers browse a catalogue of books by category, view product details, add items to a cart and place orders. Authentication and all application data are backed by Firebase: accounts are managed with Firebase Authentication, product and order data live in Cloud Firestore, and product images are uploaded to Firebase Storage.

Alongside the customer-facing storefront, the app ships with an admin panel for managing products, customers, employees and orders — making it a complete end-to-end commerce demo rather than a storefront-only sample.

State management is handled with GetX (`GetMaterialApp`, controllers and reactive theming), and the UI supports both light and dark themes.

## Features

### Storefront
- **Email & password authentication** — sign up and sign in via Firebase Authentication (`createUserWithEmailAndPassword` / `signInWithEmailAndPassword`).
- **Book catalogue** — products are loaded from the Cloud Firestore `Product` collection and presented with a category filter and a `carousel_slider` showcase.
- **Product details** — dedicated detail page per book, routed with the product id and data.
- **Shopping cart** — add books to a per-user cart backed by the Firestore `Cart` collection, with the local session id persisted via `shared_preferences`.
- **Checkout & orders** — place orders that are recorded in the `Order` / `OrderDetail` collections, with an order-placed history view.
- **Customer profile & addresses** — edit profile details and manage delivery addresses (`AddressCustomer`).
- **Light / dark theme** — reactive theme switching powered by GetX.

### Admin panel
- **Product management** — create, edit and list products, including image upload to Firebase Storage via `image_picker`.
- **Customer management** — create, edit and list customer records.
- **Employee management** — create, edit and list employees.
- **Order management** — create, edit and review orders.

## Getting started

### Prerequisites
- [Flutter SDK](https://docs.flutter.dev/get-started/install) (Dart `>=2.18.5 <3.0.0`)
- A configured [Firebase](https://firebase.google.com/) project with **Authentication** (Email/Password), **Cloud Firestore** and **Storage** enabled

### Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/DucMinhNe/EcommerceAppFlutterFirebase.git
   cd EcommerceAppFlutterFirebase
   ```

2. Install dependencies:
   ```bash
   flutter pub get
   ```

3. Connect the app to your own Firebase project. Add the platform config files generated by the Firebase console / FlutterFire CLI:
   - Android: `android/app/google-services.json`
   - iOS: `ios/Runner/GoogleService-Info.plist`

   In Firebase, enable **Email/Password** sign-in, create a **Cloud Firestore** database and enable **Storage**.

4. Run the app:
   ```bash
   flutter run
   ```
